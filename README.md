# UBeauty Retail Performance Analysis

## The Business Problem
A 19-branch retail network required an immediate analysis of one month (June) inventory and sales system export to identify revenue drivers and isolate capital tied up in dead stock.

## The Solution
I built a Python-based data pipeline using Pandas to clean a raw, unstructured system export, dynamically bypass header corruption, and aggregate financial metrics across 118 columns.

## Key Insights
* **Top Performing Location:** Sarit Centre drove the highest retail volume, generating KES 183,438. Conversely, the Online branch generated KES 0, indicating a massive digital sales gap.
* **Product Strategy:** The 'UBeauty Collagen Cream - 51g' is the highest revenue driver (KES 117,008), proving the 'Essence' category is highly lucrative.
* **Capital Risk (Dead Stock):** Identified the 'UBEAUTY youthful glow eye energe serum-15ml' as a high-risk inventory liability, with 4 units holding warehouse space but yielding zero sales network-wide.
